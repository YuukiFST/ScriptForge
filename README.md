# ScriptForge

A minimalist Windows toolkit for registry management and script conversion.

## Features
- **Compare Registry**: Compare `.reg` files with system values.
- **Generate Backup**: Create rollback files before applying changes.
- **.reg to .bat**: Convert registry files to silent batch scripts.
- **.ps1 to .bat**: Convert PowerShell to standalone batch files.

## Requirements
- Windows OS
- Python 3.9+ (if running from source)
- PyQt6 (if running from source)

## Usage
Run `ScriptForge.exe` and use the tabs to navigate between tools.
